# Logs

Log files from FileBot, FFmpeg, and other tools are stored here.

## Log Files

| File | Description |
|------|-------------|
| `movies.log` | FileBot movie rename operations |
| `tv-tvdb.log` | FileBot TV show renames (TVDB) |
| `tv-tmdb.log` | FileBot TV show renames (TMDB) |
| `sports.log` | FileBot sports rename operations |
| `courses.log` | FileBot course rename operations |

## Notes

- Log files are excluded from git via `.gitignore`
- The `.gitkeep` file preserves this directory in the repository
- Review logs to troubleshoot rename issues or verify operations
